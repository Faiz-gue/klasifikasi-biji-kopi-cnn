# Klasifikasi Citra Biji Kopi Menggunakan CNN

Project ini merupakan aplikasi klasifikasi citra biji kopi menggunakan metode Convolutional Neural Network (CNN). Dataset yang digunakan berasal dari Kaggle Coffee Bean Classification Dataset.

## Deskripsi Project

Aplikasi ini memungkinkan pengguna untuk mengunggah gambar biji kopi, kemudian sistem akan memprediksi kelas biji kopi berdasarkan model CNN yang telah dilatih.

## Kelas Prediksi

- defect
- longberry
- peaberry
- premium

## Metode

Metode yang digunakan adalah Convolutional Neural Network (CNN). CNN cocok digunakan untuk klasifikasi gambar karena mampu mengekstraksi fitur visual dari citra.
